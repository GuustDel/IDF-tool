With this tool, you can:

  • Upload existing IDF files from the SunEwat drawing tool or hand-crafted IDF files.
  • Create new IDF files from scratch when no file is available yet.

This tool is built to save time, reduce mistakes, and ensure every file meets the requirements for smooth production.
